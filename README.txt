					Instructions

Etape 1 : Vérifier que le fichier à compresser se trouve dans le même dossier que le code huffman_code.c

Etape 2 : Ouvrir le code et aller dans le terminal 

Etape 2(bis) : Attention aux lignes 483 et 484 qui génèrent l'affichage graphique de l'arbre Huffman
	       Si le fichier à compresser contient trop de caractères (harry_potter.txt), l'arbre risque
	       de saturer le terminal	 

Etape 3 : Ecrire la commande suivante dans le terminal :

		- gcc huffman_code.c -o huffman
		- ./huffman

Etape 4 : Le nom du fichier à compresser sera demandé. Il faudra donc entrer le nom du fichier. 
	  PS: Ne pas oublier d'écrire le nom du fichier avec ".txt" à la fin du nom

Etape 5 : Un fichier "compresse" se génèrera dans le dossier où se trouve le code.
 
Etape 6 : Ouvrir le fichier compressé et voilà le travail !

Etape 7 : Eteignez votre ordinateur.

Etape 8 : Si vous aviez correctement exécuté l'étape 7, vous ne seriez pas là en train de lire l'étape 8 ;)
